# 🔤 Next Word Prediction using LSTM – Deep Learning NLP Project

## 📌 Project Overview

This project focuses on building a **Deep Learning model** for **Next Word Prediction** using **Long Short-Term Memory (LSTM)** networks.  
The model learns patterns from text data and predicts the most probable next word in a sequence, which is a fundamental task in Natural Language Processing (NLP).

---

## 🎯 Objectives

- Understand sequence modeling using LSTM  
- Build a next word prediction model  
- Preprocess text data for NLP tasks  
- Train and evaluate a deep learning model for text generation  

---

## 🔧 Model Used

- Long Short-Term Memory (LSTM)  
- Embedding Layer for word representation  
- Dense Output Layer with Softmax activation  
- Loss Function: Categorical Crossentropy  
- Optimizer: Adam  

---

## 🛠️ Technologies Used

- Python  
- Jupyter Notebook  
- TensorFlow / Keras  
- NumPy  
- NLP preprocessing techniques  

---

## 🔹 Project Workflow

1. Load and preprocess text dataset  
2. Tokenize text and create sequences  
3. Convert words into numerical representations  
4. Build LSTM model architecture  
5. Train the model on sequence data  
6. Predict next word based on input sequence  

---

## 📊 Key Insights

- LSTM effectively captures **context and sequence dependencies**  
- Proper tokenization and sequence creation are crucial  
- Model can generate meaningful text predictions
