# 🔤 Next Word Prediction using LSTM – Deep Learning NLP Project

## 📌 Project Overview

This project focuses on building a **Deep Learning model** for **Next Word Prediction** using **Long Short-Term Memory (LSTM)** networks.  
The model learns patterns from text data and predicts the most probable next word in a sequence, which is a fundamental task in Natural Language Processing (NLP).

---

## 🎯 Objectives

- Understand sequence modeling using LSTM  
- Build a next word prediction model  
- Preprocess text data for NLP tasks  
- Train and evaluate a deep learning model for text generation
